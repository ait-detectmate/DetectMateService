# Installation


To install the DetectMate service, follow these steps.


Clone DetectMate repository and navigate into the repository:

```bash
git clone https://github.com/ait-detectmate/DetectMateService.git
cd DetectMateService
```

Create a virtual environment with uv:

```bash
uv venv
```

Install DetectMate and its dependencies:

```bash
uv pip install .
```
